# 介绍

这个文件夹下的示例项目是用 `xmake create -l c -t shared test` 命令生成的, xmake.lua 稍微做了一些改变

必须在纯正的 windows 操作系统下才能生成 dll 文件, 即便是在 wsl 中使用 xmake 也不会生成 dll 文件的, 只会报错
